# Cape Cod Homepage

A single-file, GeoCities-1996-style trip guide for the Short family Cape Cod trip (Chatham, MA · July 25 – Aug 1). Everything — CSS, JS, SVG badges — is inline in `index.html`. No build step, no frameworks, no external requests.

## Enable GitHub Pages

Repo **Settings → Pages → Deploy from a branch → main → / (root)** → Save. The site will be live at `https://<user>.github.io/<repo>/` in a minute or two.
